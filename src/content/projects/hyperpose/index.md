HyperPose is a library for building real-time human pose estimation systems. It combines high-level APIs for designing pose-estimation algorithms with an optimized CPU/GPU execution engine for high-resolution video streams.

The system dynamically schedules dataflow operators across heterogeneous hardware and was designed to make pose-estimation systems easier to customize, deploy, and accelerate. In evaluations, HyperPose outperformed state-of-the-art pose-estimation systems by up to 3.1×.
