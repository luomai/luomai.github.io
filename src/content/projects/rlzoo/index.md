RLzoo is a deep reinforcement learning library for designing, testing, and deploying DRL agents. It provides expressive APIs, reusable algorithms and environments, and an interactive training workflow for managing and tuning agents.

The project focuses on making reinforcement-learning systems easier to prototype and integrate into real applications, including support for custom neural-network modules and agent construction.
