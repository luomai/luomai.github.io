TensorLayer is an open-source deep learning and reinforcement learning library built on TensorFlow. It provides neural layers, pretrained models, data-processing utilities, model management, and distributed-training support for research and deployment.

The project emphasizes a transparent programming model, extensive tutorials and examples, and a broad user community. It received the ACM Multimedia Best Open-Source Software Award in 2017.
